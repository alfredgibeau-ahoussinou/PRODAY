BRIEF PRODUIT v2 — MAI 2026

ProDay est une application mobile qui relie joueurs, coachs, agents, clubs, organisateurs et sponsors sur un même écosystème football amateur et semi-pro. Stack : React Native / Expo + Firebase (Auth, Firestore, Storage).

Comptes & inscription

  1. Choix du rôle (joueur, coach, agent, organisateur, sponsor).
  2. Compte email/mot de passe ou Google / Apple.
  3. Vérification email obligatoire (comptes email/mot de passe).
  4. Profil joueur : poste, catégorie, niveau, âge, ville (filtrage Mercato).
  5. Si mineur (< 18 ans) : étape contrôle parental (responsable légal).
  6. Coach/agent : envoi diplôme ou licence → validation admin.
Protection mineurs : le contrôle parental n’apparaît que pour les comptes mineurs. Supervision, filtrage contacts, limite de temps d’écran.

Sécurité & confiance

Pour les coachs et agents :

  1. Document obligatoire à l’inscription (PDF / image).
  2. Badge orange « Vérification en cours » — messagerie limitée.
  3. Validation manuelle par administrateur.
  4. Badge vert — accès complet.

Modules produit

Mercato (recrutement)

  • Listes joueurs / clubs / coachs avec recherche.
  • Filtres : poste, catégorie (U13–Vétérans), niveau, ville, âge max.
  • Annonces, candidatures, favoris.

Matchs & saison

  • Matchs amicaux : proposer, accepter, statuts.
  • Calendrier équipe, convocations, Live Stats.

Gestion équipe (SportEasy-like)

  • Hub staff : planning, effectif, présences, stats, cotisations.
  • Rejoindre un club · convocations ciblées (sélection joueurs).
  • RSVP avec motif d'absence · pointage coach (présent / retard / excusé / absent).
  • Match amical accepté → convocation auto pour l'effectif.
  • Feuille individuelle, rapport verrouillable, export HTML/PDF, CSV cotisations.
  • Cotisation bulk pour tout l'effectif.

Arena & Sponsors

  • Tournois, inscriptions, tableau d’honneur.
  • Offres sponsors et financement club.

Profil joueur

  • Stats saison, galerie, vidéos highlights.
  • Pulse ProDay (complétion profil).
  • CV HTML → PDF exportable.
  • Menu : candidatures Mercato · contrôle parental (mineurs) · déconnexion.
