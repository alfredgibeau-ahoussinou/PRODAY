# Vision produit ProDay

Document interne — mai 2026. ProDay est l’écosystème foot amateur et semi-pro : carrière, club, matchs et communauté dans une seule application.

## Mission

**Connecter • Progresser • Réussir** — donner à chaque joueur, coach, club et organisateur les mêmes outils que le haut niveau, sans perdre des semaines dans des groupes éparpillés.

## Piliers ProDay

| Pilier | Ce que ProDay apporte |
|--------|------------------------|
| **Écosystème unique** | Recrutement, saison, convocations, matchs amicaux, Arena et sponsors — cinq onglets, une expérience |
| **Confiance** | Vérification coach / agent avant messagerie complète ; contrôle parental |
| **CV vivant** | Stats saison + export PDF alimentés par les matchs et le Live Stats |
| **Pulse ProDay** | Score de visibilité (profil, activité, confiance) — signature plateforme |
| **Mercato** | Annonces, filtres poste / niveau / ville, candidatures en un tap |
| **Saison** | Calendrier unifié, RSVP, relances, compositions |
| **Arena** | Tournois locaux, inscriptions club, tableau d’honneur |
| **Communauté** | Messagerie, détections, sponsors locaux |

## Parcours clés

1. **Joueur** — Compléter le profil → Pulse → postuler / être contacté → matchs & stats → CV PDF.
2. **Coach / agent** — Vérification → annonces & détections → convocations & Live Stats.
3. **Organisateur** — Créer tournoi → inscriptions → palmarès publié sur Arena.
4. **Parent** — Supervision, contacts, limite de temps (paramètres persistés sur le compte enfant).

## Roadmap (indicatif)

- **Livré** — Dashboard connecté, profil complet, admin, navigation inter-onglets, Mercato, matchs, convocations, Live Stats, Gestion équipe (planning, présences, stats, cotisations, annonces club), push serveur (Expo/FCM), matchs Arena MVP, contrôle parental messagerie.
- **En cours** — Déploiement Functions (plan Blaze), activation Storage.
- **À venir** — Notifications géo agents, paiements tournois.

## Ton de marque

- Parler **ProDay** et **vous / votre saison**, jamais d’autres applications.
- Mettre en avant l’**exclusif** (Pulse, hub connecté, confiance staff).
- Rester concret : actions en un tap, calendrier unifié, preuves (stats, PDF, vérification).
